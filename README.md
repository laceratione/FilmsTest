## Описание тестового задания
Необходимо создать Android приложение для отображения списка фильмов и просмотра информации о них.
Приложение должно состоять из двух экранов:
- список фильмов с фильтром
- подробная информация о фильме

Каждый экран необходимо реализовать с использованием Fragment’ов, одно Activity.

- название фильма и стрелка Назад
- обложку
- локализованное название
- список жанров и год выхода
- рейтинг
